# LLM Kernel VS Code Extension

This extension integrates with the LLM kernel and notebook API to provide a seamless user experience for managing and interacting with LLMs within Visual Studio Code.

## Features

- **Kernel Management**: Start, stop, and interact with the LLM kernel directly from VS Code.
- **Context Management**: Save, load, and reset context to maintain conversation state across sessions.
- **Notebook Integration**: Execute notebook cells and manage notebook state efficiently.
- **Custom Views**: Visualize kernel status and context in an intuitive UI.
- **Code Completion**: Get intelligent code completions based on the LLM kernel's capabilities.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/vscode-llm-kernel-extension.git
   ```
2. Navigate to the project directory:
   ```
   cd vscode-llm-kernel-extension
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open the project in Visual Studio Code:
   ```
   code .
   ```
5. Run the extension:
   - Press `F5` to start a new Extension Development Host instance of VS Code.

## Usage

- Use the command palette (`Ctrl+Shift+P`) to access LLM kernel commands.
- Manage context and kernel status through the provided views.
- Customize settings in the extension configuration.
